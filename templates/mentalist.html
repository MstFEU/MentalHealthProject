What is the Mentalist ? " The Mentalist " is an analytical website powered by Machine Learning algorithm, in order to analyze users mental health and to predict propability of their mental issues risk.
____The goal of this web developer is to raise people awareness on the importance of mental health and provides chance to avoid mental issues, since the mental health problems are tending to be major trouble which presently hits people hard, such problems need to be seriously concerned. ____This website is created as a medical project with machine learning algorithm for the Cooperation Education of the electrical engineering of Biomedical Engineering Institute, Chiang Mai University (BMEI).
____Therefore, the purpose of this website is for only educational sake and not for any commercial benefit at all.
Working Process of the Mentalist
1. Datapreprocessing

Data Cleaning and Studying

Datapreprocessing is the process of preparation and cleaning of raw dataset on mental illness imported from Kaggle, the online datascience community and largest site for public datasource. The Mentalist requires prepared dataset as training and testing dataset of its embedded machine learning algorithm.
2. Machine Learning Algorithm

Writing an Analytical ML System

The heart of the Mentalist is its Artificial Intelligent analyzer. The system learns to analyze and predict mental health by its system relied on the algorithm named Random Forest Classifier, ensemble learning method for classification, regression which powered by Scikit-Learning Module.
3. Responsive Website

Create Website for Users

In order to analyze a mental health of an individual user, it requires an interface which fetches data from user into analytical algorithm. the Mentalist responsive website is created in order to provides users accessibility to analyze their mental health on our machine learning system.
